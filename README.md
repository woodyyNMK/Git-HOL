# 📘 My GitHub Project

## 🔹 Description
This is a simple GitHub project created for practicing **Git & GitHub collaboration workflows**.  
The project demonstrates how to:
- Create and manage branches
- Add new features
- Use pull requests
- Collaborate with teammates

---

## 🚀 Getting Started

### Prerequisites
- [Git](https://git-scm.com/downloads) installed
- A GitHub account

### Steps 
```bash
# Clone this repository if your are already a collaborator of this repo
git clone https://github.com/woodyyNMK/Git-HOL.git

# if not a collaborator yet, fork this repo and clone that new forked repo.

# Navigate into the folder
cd Git-HOL

# Create a new branch for your work
git checkout -b your-new-branch

# Start creating files, modifying the existing file

# Then commit and push
git add .
git commit -m "Your commit Message"
git push origin your-new-branch

# Do PR
